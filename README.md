## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Use the calculator by clicking the buttons or using your keyboard

## Keyboard Shortcuts

- Numbers: 0-9
- Operations: +, -, *, /, %
- Calculate: Enter or =
- Clear: Escape
- Backspace: Backspace key
